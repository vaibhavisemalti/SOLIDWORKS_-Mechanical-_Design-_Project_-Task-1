# SOLIDWORKS_-Mechanical-_Design-_Project_Task-1
This repository contains my SOLIDWORKS Mechanical Design Project Task 1. It includes the design of a basic mechanical component using 2D sketching and feature-based 3D modeling. The project uses Extrude Boss/Base, Extruded Cut, Fillet, Hole Wizard, and proper dimensioning following standard engineering design practices.
# Objectives
 Learn the SOLIDWORKS user interface.
- Create and fully define 2D sketches.
- Build a basic mechanical component using Extrude Boss/Base, Extrude Cut, Fillet/Chamfer, Hole Wizard.
- Apply engineering dimensions and design features correctly.
- Organize the feature tree following good CAD practices.
  # Software used
  SOLIDWORKS
  # PROJECT FILES
  This repository includes:
- SOLIDWORKS Part File (.SLDPRT)
- Drawing File (.SLDDRW)
- Project screenshots
- Project report 
# Learning outcome
This project helped me to understand the basic fundamentals of SOLIDWORKS, including sketch creation, feature-based modeling, dimensioning, and proper model organization. It also improved my understanding of basic mechanical designs.
